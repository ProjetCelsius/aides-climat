## Résumé du projet

Version open source / publique du moteur d'éligibilité aides climat. Application web (Vite + vanilla JS). 47 dispositifs, recherche par SIREN.

**Repo public** : destiné à être partagé / démontré
**URL cible** : aides-climat.fr (à confirmer)
**Stack** : Vite + JS vanilla

## Relation avec celsius-eligibilite
Le repo `celsius-eligibilite` (private) est l'outil interne de prospection avec dashboard commercial, funnel mail, PDF. Ce repo (aides-climat) est une version simplifiée publique du même moteur.
